## 🚀 Featured Projects

### 🤖 [Zalo ChatGPT Bot](https://github.com/IamAnhKhoa/zalo-chatgpt-bot)
> A Zalo messaging bot integrated with ChatGPT, context-aware via Google Sheets knowledge base

- **Stack:** Next.js · Vercel · Google Sheets API · ChatGPT2API (Render.com)
- **Architecture:** Zalo User → Webhook (Vercel) → AI → Google Sheets
- **Features:** Persistent chat history, custom knowledge base Q&A, one-click Vercel deploy

```
Zalo User → Webhook → ChatGPT2API → Google Sheets
```

---

### 🧾 [VAT Invoice Bot — Telegram](https://github.com/IamAnhKhoa/bot-hoadon)
> Telegram bot that reads invoice photos with AI and automatically logs them to Google Sheets

- **Stack:** Cloudflare Workers · Anthropic Claude Haiku · Google Sheets API
- **Cost:** ~$0.10/month for 100 invoices/day 💰
- **Features:** Photo input → AI extraction → Automatic sheet logging, 100% serverless

```
Telegram → Cloudflare Worker → Claude Haiku → Google Sheets
```

---

### 🏥 [Healthcare HR Management System (HRM)](https://github.com/IamAnhKhoa/hrm-webapp)
> Web app for managing healthcare personnel in public medical facilities

- **Stack:** Next.js · Supabase · Zalo OA API · Vercel
- **Features:** Staff profile management, workforce reporting, scheduled Zalo reminders via cron
- **Data:** Excel import, role-based RLS access control, Zalo Bot integration

---

### 🛍️ [Button & Accessories E-Commerce Website](https://github.com/IamAnhKhoa/khuynut-website)
> E-commerce catalog website for buttons and garment accessories

- **Stack:** Next.js · Vercel · Static Export
- **Features:** Product catalog, WebP image optimization, SEO-friendly
