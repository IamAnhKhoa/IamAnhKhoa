## 🙋‍♂️ Giới thiệu

Tôi là một lập trình viên làm việc trong lĩnh vực **y tế & hành chính công**, đam mê xây dựng các công cụ tự động hóa giúp giảm tải công việc giấy tờ và tăng hiệu quả vận hành. Tôi thích khám phá AI, chatbot và các giải pháp không máy chủ (serverless).

## 🚀 Dự án nổi bật

### 🤖 [Zalo ChatGPT Bot](https://github.com/IamAnhKhoa/zalo-chatgpt-bot)
> Bot Zalo tích hợp ChatGPT với ngữ cảnh dữ liệu từ Google Sheets

- **Stack:** Next.js · Vercel · Google Sheets API · ChatGPT2API (Render.com)
- **Kiến trúc:** Zalo User → Webhook (Vercel) → AI → Google Sheets
- **Tính năng:** Lưu lịch sử chat, trả lời theo knowledge base tùy chỉnh, deploy Vercel

```
Zalo User → Webhook → ChatGPT2API → Google Sheets
```

---

### 🧾 [Bot Hóa Đơn VAT — Telegram](https://github.com/IamAnhKhoa/bot-hoadon)
> Bot Telegram đọc ảnh hóa đơn bằng AI, tự động ghi Google Sheets

- **Stack:** Cloudflare Workers · Anthropic Claude Haiku · Google Sheets API
- **Chi phí:** ~$0.10/tháng cho 100 hóa đơn/ngày 💰
- **Tính năng:** Nhận ảnh → AI phân tích → Ghi sheet tự động, serverless 100%

```
Telegram → Cloudflare Worker → Claude Haiku → Google Sheets
```

---

### 🏥 [Hệ thống Quản lý Nhân sự Y tế (HRM)](https://github.com/IamAnhKhoa/hrm-webapp)
> Web app quản lý nhân sự dành riêng cho đơn vị y tế công lập

- **Stack:** Next.js · Supabase · Zalo OA API · Vercel
- **Tính năng:** Quản lý hồ sơ viên chức, báo cáo nhân lực, cron job nhắc việc qua Zalo
- **Dữ liệu:** Import Excel, RLS theo phân quyền, tích hợp Zalo Bot

---

### 🛍️ [Website Khuy Nút](https://github.com/IamAnhKhoa/khuynut-website)
> Website thương mại điện tử cho sản phẩm khuy nút, phụ kiện may mặc

- **Stack:** Next.js · Vercel · Static Export
- **Tính năng:** Catalog sản phẩm, tối ưu ảnh WebP, SEO-friendly
